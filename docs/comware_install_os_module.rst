comware_install_os
++++++++++++++++++

.. contents::
   :local:
   :depth: 1


Synopsis
--------

Added in version 1.8

Offers ability to copy and install a new operating system on Comware v7 devices.  Supports using .ipe or .bin system and boot packages.
